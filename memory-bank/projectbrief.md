# Project Brief

## 项目名称
nonebot-plugin-jmdownloader

## 项目概述
这是一个基于 NoneBot2 框架的 JMComic 漫画下载插件，支持在 QQ 群聊和私聊中搜索、查询和下载 JMComic 网站上的漫画内容，并支持将下载的 PDF 文件上传到群文件夹。

## 核心目标
1. 提供便捷的 JMComic 漫画搜索和下载功能
2. 支持群聊和私聊两种使用场景
3. 提供完善的内容过滤和用户管理机制
4. 支持自定义配置以适应不同使用需求

## 核心需求

### 功能需求
- **搜索功能**: 通过关键词搜索漫画，支持分页显示
- **查询功能**: 通过 JM 号查询漫画详细信息
- **下载功能**: 下载指定漫画为 PDF 并支持上传到群文件夹
- **用户管理**:
  - 用户下载次数限制（每周重置）
  - 群黑名单管理
  - 群功能启用/禁用控制
- **内容过滤**:
  - 支持屏蔽特定 JM 号
  - 支持屏蔽特定标签
  - 预置常见敏感标签

### 非功能需求
- 仅支持 OneBot V11 协议
- 支持代理配置
- 支持多线程下载
- 支持 JMComic 账号登录
- 定时清理缓存文件

## 目标用户
- NoneBot2 机器人运营者
- 需要在 QQ 群中提供漫画下载服务的群主/管理员

## 技术约束
- Python 3.10+
- NoneBot2 2.4.2+
- OneBot V11 适配器
- 依赖 jmcomic 库进行漫画下载

## 项目范围
- ✅ 漫画搜索、查询、下载
- ✅ 用户权限和次数管理
- ✅ 内容过滤机制
- ✅ 群文件夹管理
- ❌ 其他协议适配器支持（仅 OneBot V11）
- ❌ 在线阅读功能
